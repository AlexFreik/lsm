YAD: YouTube Auto Delay

GitHub

This website allows you to configure the delay for a YouTube Live Stream to have a fixed delay. Please check the bottom of this page for more details.

Example 1: The configured delay is 15min and the current delay is 13min 59s, it will be reset back to 15min. And if the current delay is 1h 15min 1s, it will be also reset back to 15min.

Example 2: The configured delay is 15min, but due to network issues the delay slowly become is 20min. Now if YT skips and delay becomes 0s the script will reset delay back to 20min 5s.
